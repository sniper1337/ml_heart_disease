# Сравнение моделей для прогнозирования сердечных заболеваний

## Обзор

В данном отчете представлен сравнительный анализ четырех моделей машинного обучения для прогнозирования сердечных заболеваний. Модели оценивались с использованием кросс-валидации и отложенной тестовой выборки для оценки производительности, стабильности и способности к обобщению.

## Основные выводы

### Лучшая модель в целом: **KNN**

| Показатель | Значение |
|------------|----------|
| Test Accuracy | **0.9016** (лучший) |
| Test F1 Score | **0.9032** (лучший) |
| Test ROC-AUC | 0.9583 (2-й) |
| Время обучения | **0.07 сек** (самый быстрый) |
| CV ROC-AUC | 0.8908 ± 0.0257 |

**KNN** достигает наилучшего баланса между точностью, полнотой и вычислительной эффективностью.

## Рекомендации

### Для production-среды: **KNN**
- ✅ Высокая точность (90.16%)
- ✅ Отличный баланс F1 (0.9032)
- ✅ Минимальное время обучения (0.07 сек)
- ✅ Хорошая стабильность

### Для задач, критичных к ROC-AUC: **CatBoost**
- ✅ Лучшая дискриминационная способность (0.9643)
- ❌ Долгое обучение
- Рекомендуется, если важнее минимизировать ложные срабатывания

### Для интерпретируемости: **Logistic Regression**
- ✅ Самая стабильная модель
- ✅ Хорошая интерпретируемость коэффициентов
- ✅ Быстрое обучение
- ❌ Самая низкая точность среди конкурентов

##  Заключение

**Рекомендуемая модель для внедрения: KNN**

Модель KNN показывает наилучшие результаты по большинству метрик (Accuracy, F1), при этом обучается практически мгновенно. CatBoost может быть рассмотрен как альтернатива, если критически важна максимизация ROC-AUC (например, для медицинского скрининга), но с учетом значительно большего времени обучения.
